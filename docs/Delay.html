Namespace: Delay

Delay

Aguarda tempos determinados para executar funções.

Cada tick corresponde 500ms.

Author:
  • Arthur L
  • Gabriel T

Methods

(static) cancel(task)

Cancela o delayScript que for passado.

Parameters:
Name Type Description
task DelayTask

Wait/Delay a ser interrompido.
Example
const delay = Delay.wait(() => {
     Engine.debug('Teste')
}, 10)

Delay.cancel(delay)
//Função não executará, pois o delay foi interrompido antes do tempo a ser aguardado.

(static) interval(callback, ticks) → {DelayTask}

Executa uma função no intervalo de tempo.

Parameters:
Name Type Description
callback function

Função executada sempre que o tempo determinado ter passado.
ticks Number

Quantidade de ticks a aguardar até a execução da função.
Returns:
Type
DelayTask
Example
Delay.interval(() => {
 //Executado a cada 1 segundo.
}, 2)

(static) seconds(sec) → {Number}

Converte segundos em uma quantia de ticks correspondente.

Parameters:
Name Type Description

Quantidade de segundos a serem convertidos em ticks.
Returns:
Type
Number
Example

}, Delay.seconds(10))
